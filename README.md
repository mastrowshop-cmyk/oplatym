# Telegram Bot для Oplatym.ru

Бот-модератор с автоответами и приветствиями.

## Функции
- Модерация чата
- Приветствие новых участников  
- Автоответы на ключевые слова
- Защита от мошенников

## Установка
1. Клонируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Замените BOT_TOKEN в bot.py
4. Запустите: `python bot.py`

## Команды
- /start - информация
- /help - помощь
- /rules - правила
